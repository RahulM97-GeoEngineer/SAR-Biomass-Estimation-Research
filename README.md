
# SAR Biomass Estimation Research

## Overview

This repository contains research workflows, documentation, and Python tools for crop biomass estimation using Sentinel-1 SAR and Sentinel-2 optical data.

## Research Domain

- Remote Sensing
- SAR Remote Sensing
- Precision Agriculture
- Machine Learning
- GIS

## Study Area

Ahmedabad District, Gujarat, India

## Datasets

- Sentinel-1 GRD
- Sentinel-2 MSI
- SRTM DEM

## Software

- ESA SNAP
- Python
- QGIS
- ENVI

## Methodology

1. SAR Preprocessing
2. Optical Preprocessing
3. Feature Extraction
4. Water Cloud Model
5. Machine Learning
6. Validation
7. Biomass Mapping

## Research Status

Ongoing M.Tech Research
